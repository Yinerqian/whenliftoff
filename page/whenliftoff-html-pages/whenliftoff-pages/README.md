# whenliftoff HTML 拆分页面

本文件夹根据附件模拟图拆分为 4 个静态 HTML 页面：

1. `launch-list.html`：火箭发射列表 / 发射日程页
2. `launch-detail.html`：火箭发射详情页
3. `news-list.html`：航天新闻列表页
4. `news-detail.html`：航天新闻详情页

共享样式：`styles.css`
图片素材：`assets/`

字体设置：
- 英文字母：Space Grotesk，通过 `.latin` 类控制。
- 中文与数字：Noto Sans SC，通过页面默认字体和 `.num` 类控制。

当前为静态页面，后续开发时可将任务数据、倒计时、新闻数据替换为 API 返回内容。
